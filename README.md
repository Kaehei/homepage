## 🚀 快速开始

<details open>
<summary><strong>安装依赖</strong></summary>

```bash
npm install
```
</details>

<details open>
<summary><strong>启动开发环境</strong></summary>

```bash
npm run dev
```
</details>

<details open>
<summary><strong>访问项目</strong></summary>

[http://localhost:3000](http://localhost:3000)
</details>
